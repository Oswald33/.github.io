Development difficulties

1-The room’s design

The room‘s difficulty is how to deal with the player’s movement, I gave several of colliders to the room(include 2D box collider and edge collider), to make sure the path of the player is right.

2-The stairs design

The stairs contain three components, the sprite and the edge collider make up the stairs, and we need a script code to control it, make sure the Player run on the right path.

3-The Player design

The player has three colliders, two box colliders and one circle collider, the circle collider keep the Player stand on the ground, the center 2d box collider will collide with the coloring door, when the player passed the coloring door, the player will change the color of itself, and the last collider is acting on the wall, it decides the player’s movement direction.
