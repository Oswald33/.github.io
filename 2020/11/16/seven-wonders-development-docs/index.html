Game Implement Process

1-Box Level

The box level is the core gameplay of this game. So we should ensure the interactivity and scalability to support box triggers of any quantity.

2-Player Controller

The Player Controller class offers movement, jumping and double jumping. And in this game, the jump dust will make the destructible platform to be destroyed, so I put the jump dust as a single class.

3-Enemy Movement

The enemies in the air are ghosts. A ghost carries a box and move along a curve, and it will hurt the player. So in the following code, I gave some parameters to it, such as: damage, routes and the health point.

Development difficulties

2-Destructible Platform Implementation

I used Unity 2D tile map to design the destructible platform. When the player jumps, the tile will be destroyed.

3-Enemy Movement

By using the bezier curve formula, I get a curve in Unity, which enemies will move along. At the same time, I can merge two curves into one so that enemies can go further.
